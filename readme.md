# **Simple video compressor using ffmpeg**

## Usage:

### install all requirements using pnpm

*You can use whatever packagemanager you want, but i prefer pnpm*

```bash
pnpm install
```

### run the dev mode

```bash
pnpm run dev
```

### build the app

```bash
pnpm run build-mac
pnpm run build-linux
pnpm run build-win
```

generated app will be in the dist folder
